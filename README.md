## About

This portfolio is a single-page application built using Next.js. It includes sections for an introduction, skills, projects, and contact information. The goal is to provide a clean, modern, and responsive design to effectively showcase my work and skills.

## Features

- Responsive design for mobile and desktop
- Dynamic content loading with Next.js
- Smooth scrolling navigation
- Projects gallery with links to live demos and repositories
- Contact form with email integration
- SEO optimized

## Technologies Used

- Next.js
- React
- CSS Modules / Tailwind CSS
- Email.js (for contact form)
- Vercel (for deployment)

## Getting Started

To get a local copy up and running, follow these steps.

### Prerequisites

- Node.js
- npm or yarn

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/yourusername/your-portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   # or
   yarn install
   ```

### Running the Development Server

1. Run the development server
   ```sh
   npm run dev
   # or
   yarn dev
   ```
2. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Usage

Navigate through the website using the navbar or by scrolling. The projects section contains cards with links to live demos and GitHub repositories. Use the contact form at the bottom to send me a message.
